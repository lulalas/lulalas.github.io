我的第一篇博客

Posted on 2020-04-16 Edited on 2020-04-30

Windows10搭建个人博客（hexo&GitHub）

1.安装Node.js。

安装包下载地址：nodejs.org

下载左边的LTS(Long Term Support) 安装。直接双击安装一直下一步就可以了（当然安装路径可以自己改一改）。

Ps:我下载的时候网速很慢很慢，不是网络的原因，应该是网页或者浏览器的原因。慢慢来不要急。。。(*￣︶￣)
安装好之后win+R，输入cmd打开命令管理器，输入node -v 查看版本,有版本显示安装成功。

2.安装hexo

首先我们需要先来安装个cnpm提高速度:
npm install -g cnpm --registry=http://registry.npm.taobao.org
以后下载什么东西都可以用cnpm,关于npm与cnpm的区别参考：https://www.cnblogs.com/yanggb/p/10822420.html

然后安装hexo：cnpm install -g hexo-cli

查看版本，测试一下是否安装成功：hexo -v

然后我们需要创建一个blog的目录，用来存放个人博客相关的东西：mkdir blog
进入创建的目录：cd blog
初始化博客：hexo init
启动博客本地服务：hexo s

通过http://localhost:4000本地访问我们所创建的博客，就是这个样子的

Ctrl+C停止运行

3.安装git

安装git可以直接在命令行进行安装，要按照在blog目录下，命令是：
cd blog
cnpm install --save hexo-deployer-git
但是用这种方法可能下载的比较慢，如果下载过慢可以使用以下的链接：
https://npm.taobao.org/mirrors/git-for-windows/

选择合适的版本下载，然后手动安装在blog目录下，安装直接点下一步就可以了。安装完之后可以在这里看到，这样就安装好了：

5.部署代码到远端&完成

保存好之后回到blog目录。右键点击创建的blog目录，选择Git Bash Here，将代码推到GitHub上：hexo d

在这个过程中需要输入GitHub账户和密码。
完成后就可以使用https://yourname.github.io/来访问你的博客啦
至此，个人博客搭建成功!下一篇再写博客的美化和内容。。。。。。

几个可能出现问题的地方：

1.创建仓库的时候名字一定要是yourname.github.io。不然会出现404的问题，如果开始名称写错可以Setting修改。

2.运行hexo d命令是可能会出现以下错误：fatal: unable to auto-detect email address
此时我们需要打开.git文件中的config文件进行配置
